# 🩺 Diabetes Prediction using Decision Tree Classifier

An end-to-end Machine Learning project that predicts whether a patient is diabetic using a **Decision Tree Classifier**. This project follows the complete machine learning pipeline, including data preprocessing, exploratory data analysis (EDA), model training, hyperparameter tuning, prediction, and performance evaluation.

---

## 📖 Project Overview

Diabetes is one of the most common chronic diseases worldwide, making early detection essential for effective treatment and prevention of complications.

This project uses the **Decision Tree Classification** algorithm to analyze patient health data and predict the likelihood of diabetes based on various medical attributes.

The notebook demonstrates the complete workflow of building a supervised machine learning classification model using Python and Scikit-learn.

## 🗂️ Dataset Information

The project uses the **Pima Indians Diabetes Dataset**, which contains diagnostic measurements collected from female patients.

### Features

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI (Body Mass Index)
- Diabetes Pedigree Function
- Age

### Target

- **0** → Non-Diabetic
- **1** → Diabetic
